# interview-flashcards
Interview Flashcard App: https://flashcards-dj.herokuapp.com/
## Overview

Using Material UI/React, I'm building a simple CRUD app to compile questions/answers that could come up during technical interviews â€” including behavioral components, technical trivia (specifically JavaScript), and system design questions.
